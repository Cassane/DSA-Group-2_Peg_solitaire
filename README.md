# DSA-Group-2_Peg_solitaire
Peg Solitaire – 9x9 Version with Power-Ups

This project is a modified version of the classic Peg Solitaire game. It introduces new features to increase strategic depth and replayability, while keeping the game fully implemented in Python.

Features

9×9 cross-shaped board

Normal pegs (O) and power-up pegs (P)

Blocked holes (X) that normal pegs cannot jump over

Normal jumps (over 1 peg) and long jumps (over 2 pegs)

Power-up pegs can jump over blocked holes

Randomized board layout for each game

Recursive backtracking solver to automatically find a solution
